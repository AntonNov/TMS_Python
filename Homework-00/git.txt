cd path - заходим в ту папку куда будем клонировать репозиторий
git clone https://github.com/AlexandrSech/Z63-TMS.git - клонируем наш репозиторий
cd Z63-TMS - переходим в клонированый репозиторий
ВАЖНО! Все работы с гитом делать из корня репозитория, в нашем случае Z63-TMS
git checkout имя_вашей_ветки - переходим в вашу ветку и работаем только из нее
Далее делаем то что хотим, вносим изменения, создаем/удаляем объекты
git add . - добавляем все новые файлы и папки в git
git commit -am "текст нашего коммита"
git push -u origin имя_вашей_ветки

Переходим на сайт гита и в нашем репозитории создаем pull request, чтобы ваши изменения попали в ветку main

После того как администратор подтвердил ваш pull request вы можете работать дальше

git checkout main - переходим обратно в ветку main, чтобы скачать последние версии всех файлов и папок
git pull - скачиваем

Повторять все кроме первых двух строчек
